# PasserGen
An ultimate tool for Password Generation, having the ability to generate passwords 8-24 characters long, have the ability to test out password strength, and having the ability to generate custom passphrases.   

GENERATE PASSWORD -
This section allows you to generate the actual password, you can edit the character size of the password for 8-24 characters. You also have a section that allows you to put in custom characters. You also get the option to deselect ambiguous characters such as l, 1, I, O, and 0

TEST YOUR PASSWORD -
This section allows you to insert your password to test it's strength. It tests your strength through 5 different levels. Very Weak, Weak, Medium, Strong and Very Strong. This pairs with Password Entropy which tells you through statistics. The Estimated Crack Time runs an algorithm, testing through lots of different custom character sets, then tells you how long it would take for an algorithm or person to crack it. And finally, the Pwned Password Check, telling you whether or not your password was evver found in any data breaches.

GENERATE PASSPHRASE -
This section allows you to generate passpphrases out of easy to remember spanish and english words. You can choose between 3 through 8 words. You can choose the seperator for the words, capitilize the first letter of each word, and use custom words. 


EXTRAS -
Top right corner has the CAPTCHA Practice. It is usable, and you msut select all the emojis it tells you to. It is VERY unfinished and can even break the code sometimes, so use it with caution.
Within the Test Password Strength section, it will give you some security tips if your password isn't strong enough.
There is a password history bar at the bottom of the page, if you accidentally generate another password, it will arrive there, allowing you to still copy it.
Also at the bottom of the screen it reminds you of TFA (Two Factor Authentication)
